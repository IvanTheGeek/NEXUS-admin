# NEXUS-admin

> Workspace instructions and references: `../AGENTS.md`, `../FSHARP.md`, `../WORKFLOW.md`

## Purpose

Admin hub for the NEXUS workspace. Stores project templates, setup checklists, and workspace-wide preferences used when bootstrapping a new NEXUS project.

Reference this project when:
- Starting a new NEXUS project — copy from `templates/new-project/` and follow `setup/checklist.md`
- Updating workspace-wide scaffolding standards — templates here are the source of truth

## Contents

- `templates/new-project/` — files to copy verbatim when creating a new project (update placeholders after copying)
- `setup/checklist.md` — step-by-step bootstrap guide for a new NEXUS project

## Stack

No F# solution — documentation, templates, and scripts only.
